## 📖 Overview

This repository powers the personal website served at [ahmetbostanciklioglu.github.io](https://ahmetbostanciklioglu.github.io). It is a set of static HTML pages, styled inline with no build step or dependencies. The home page links to per-app privacy policies and support pages for a collection of privacy-first indie iOS apps, each hosted as its own standalone HTML page.

## ✨ Features

- Single static landing page (`index.html`) with a card-based directory of apps, styled with system fonts and gradient app-icon placeholders.
- Dedicated privacy policy pages for each app: GiftAI, CoinScan, ReceiptTax, AntiqueAI, SlabGrade, DogLens, and Habity.
- A support page for Habity (`habity-support.html`) with an FAQ section and a contact call-to-action.
- Pure HTML and inline CSS — no frameworks, no JavaScript, no build tooling required.
- Responsive layout with a mobile-friendly viewport and hover interactions on app cards.
- Served directly via GitHub Pages from the repository root.

## 🚀 Getting Started

Clone the repository and open any page directly in a browser — no build step or dependencies are needed.

```bash
git clone https://github.com/ahmetbostanciklioglu/ahmetbostanciklioglu.github.io.git
cd ahmetbostanciklioglu.github.io
open index.html   # or just double-click the file
```

To preview it as it would appear when served, run any static file server, for example:

```bash
python3 -m http.server 8000
# then visit http://localhost:8000
```

## 📋 Requirements

- Any modern web browser to view the pages.
- No build tools, package managers, or runtime dependencies.
- A GitHub Pages-enabled repository (served from the default branch) to publish.
